Careers in the Computer Science and Engineering field

Software Developer Software Developers build computer programs or mobile apps independantly. They desingn, test, fix and debug themselves before sending their finished product to it's destination. This is very difficult becuase one must have the creativity to design the user interface, and the skills to make the program easy and nice to use. The job is extremely diffucult, and the salary reflects that, software developers get paid anywhere from $50 000 to $250 000.
Computer Hardware Engineer Hardware Engineers are tasked with creating and testing their own components and making sure their products are safe and up to date. Hardware engineers build components such as motherboards, graphics cards, memory devices. The avererage salary for a computer hardware engineer is around $100 000 to $200 000 /year.
Computer Network Architect Network architectes are responsible for designing, creating and implementing computer networking solutions such as LANs, WANs, servers etc. They must compile the software and hardware to make the network possible and implement their solutions in a large scale. Network architects must be well trained in hardware, the internet, and software. Salaries range from $100 000 to $200 000 /year.
Web developer Working purely with software, web developers require mastery in languages such as PHP, Javascript, HTML and CSS. Web developers are responsible for designing, creating and deploying their web application or service. Some examples of web developer products include Twitter, Facebook and Instagram. Salaries range from $35 000 to $120 000
Computer Programmer Writes the "backend" code which provides all of the fuctionality to the user. Programmers are tasked with writeing the code in the first place, testing the code under high stress and other factors, and publishing the code to it's destination. Common languages are C#, C++, Swift 3, Python etc. Computer programmers make anywhere from $45 000 to $120 000 per year.
